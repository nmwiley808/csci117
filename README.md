## Course Description

CSCI 117 examines the **structure, design, and implementation of programming languages**. The course explores fundamental language concepts and compares different programming paradigms used in modern computing.

Topics covered in this course include:

- Programming language paradigms  
- Scope and binding rules  
- Language semantics and syntax  
- Functional programming concepts  
- Logic programming concepts  
- Concurrency and declarative programming  
- Language implementation and interpretation  
- Applications of programming language theory  

The course emphasizes understanding **how programming languages are designed and how different paradigms influence program structure and software development**.

---

## Programming Languages Used

Programming assignments in this course are implemented using:

- **Haskell** – functional programming language  
- **Oz** – multiparadigm language supporting functional, logic, and concurrent programming  
- **Prolog** – logic programming language based on formal logic

These languages help demonstrate different programming paradigms and computational models.
